# 📚 Help Me Read

> [English version](README_EN.md)

把学术文献变成**可学习的课程**和**可检索的知识库**。

读论文常有两个痛点：读完就忘（没真正学懂）、读了没沉淀（没变成自己的知识）。Help Me Read 针对两者——先生成一套循序渐进的 HTML 课程带你学懂，再产出结构化笔记，配合术语原子笔记、待学习清单、溯源式问答，让一篇文献真正进入你的知识网络。

> 💡 **推荐配合 [Obsidian](https://obsidian.md/) 使用**，以获得完整的知识库体验（结构化笔记、术语原子笔记、双向链接）。没有 Obsidian 也可以独立使用课程功能，同样能帮你学懂文献。

## ✨ 功能

### 🎓 课程 —— 把文献变成可学习的若干节课

- 将文献按学习路径重组为**若干节**循序渐进的课程，每节聚焦 1-2 个核心概念
- 对新手指南友好，每节标注"可跳过条件"，有基础可跳过
- 每节课包含：学习目标、核心讲解、代码实现（可选，有可信源码时才出现）、自测小问题、本课小结
- 课程以独立 HTML 文件呈现，在浏览器中即可学习，不依赖任何特定软件

### 📝 笔记 —— 写入结构化知识库

- 根据文献类型自动选用对应的总结结构：
  - **研究论文** → 背景与动机 / 问题定义 / 方法（含公式）/ 实验 / 结论与局限 / 相关工作
  - **综述** → 领域概述 / 分类体系 / 方法对比 / 发展趋势 / 开放问题
  - **数据集论文** → 构建动机 / 数据构成 / 标注流程 / 统计分布 / 基准评测 / 局限与伦理
- 每一条总结都标注原文来源【页码 / 章节】，可溯源
- 关键公式用数学符号重现并解释含义
- 术语自动标注为候选原子笔记，学完后引导你亲手拆分（亲手拆知识才能内化）
- **待学习机制**：遇到文献没解释的概念，可存到待学习目录，并协助搜索相关资料
- **自测模块**（默认关闭）：在笔记末尾添加问答式题目，你填写理解后进行严格评审

### 💬 问答 —— 基于原文的精准答疑

- 力求一语中的，问什么答什么，不绕弯、不过度扩展
- 能引用原文的尽量先引用原文，再补充必要的简要解释
- 每条回答标明信息来源和原文出处
- 文献范围外的问题如实告知"原文未涉及"，不臆造

## 🚀 使用

通过 `/help-me-read` 指令或自然语言均可触发：

```
/help-me-read https://arxiv.org/abs/1706.03762
帮我读一下这篇 https://arxiv.org/abs/1706.03762
总结这篇 PDF：D:\papers\transformer.pdf
帮我读一下 Attention Is All You Need
把这篇做成课程，我是新手
这篇综述讲了什么
```

也可以用子命令指定局部执行——说"只总结"仅生成笔记，说"做成课程"仅生成课程，直接提问则进入问答。

如果只给出论文标题（没有 URL 或 PDF），agent 会联网搜索并找你确认后再继续。

### 📂 Obsidian vault 路径

首次使用时询问 vault 路径，之后自动保存到配置文件 `~/.help-me-read.json`，下次无需再输入。也可以手动编辑该文件修改路径：

```json
{
  "obsidian_vault": "D:\\Path\\To\\Vault"
}
```

## ⚙️ 配置

| 项 | 默认 | 说明 |
|---|---|---|
| Obsidian vault 路径 | 询问 | 首次使用询问后持久化到 `~/.help-me-read.json`，也可手动编辑 |
| 自测模块 | 关闭 | 开启后，笔记末尾会添加问答式自测题（3-5 题），填写理解后进行严格评审——准确则通过，不准确则指出偏差让你重填，不直接给答案（除非你明确要求） |
| 问答记录保存 | 不保存 | 每轮问答后询问"是否需要保存这次问答"，选择保存则单独存为独立笔记，不会混入文献总结笔记中 |
| 课程节数 | 3-5 节 | 按文献篇幅自动决定，短文少节，长篇多节；硬上限 5 节 |
| 代码实现 | 自动 | 当论文涉及算法/模型且有可信源码时，自动在课程中追加对应代码片段（附带逐行注释），无可信源码则不出 |
| 课程主题 | 浅色 | 默认浅色。说"用深色"切换。偏好持久化到 `~/.help-me-read.json` |

## 📊 支持的文献类型

| 类型 | 总结结构 | 课程形态 |
|---|---|---|
| 研究论文 | 学术标准结构 | 学习路径重组 |
| 综述 | 领域地图结构 | 全景 + 流派逐节 |
| 数据集论文 | 数据生命周期结构 | 数据生命周期逐节 |
| 其他（案例研究/技术报告/观点论文等） | 归入研究论文模板，灵活调整章节标题 | 学习路径重组 |
